1.-Programación Orientada a Objetos con php - Clases con php

	Las clases son un espacio de trabajo donde podemos usar funciones 
	y atributos ,dentro de php podemos declarar N clases 

	se declara con la palabra reservada "class" seguido del nombre de la clase
	y sus llaves 

			class miClase{
				
				#esto es una clase 
			}

2 .-Programación Orientada Objetos con php - Métodos y atributos con php

	Los metodos es un conjunto de instrucciones que realizaran una tarea
	en especifica y estos se declaran con el modificador de acceso "public"
	y la palabra reservada "function"
			

			public function miMetodo(){
					
				#esto es un metodo
			}


	lo satributos son variables que podemos usar en una clase y estos llevan
	el modificador de acceso, ejemplo 
		
			public $atributo ="";
